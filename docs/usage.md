# Friend友谊服务器玩法手册

## ⚙️ 核心指令功能

### `/cd`：主菜单（核心入口）

`/cd` 是服务器功能总入口，输入后打开交互菜单，集成以下实用功能，点击对应选项即可使用：
- **名称配色**：切换游戏内ID栏、聊天框昵称的显示颜色（仅可切换已获得的配色）；
- **社团管理**：查看已登记社团列表、申请加入社团、查看自身社团信息（成员、公告等）；
- **CoreProtect查询**：图形化界面快速查看方块操作记录（如被破坏建筑的操作人、时间），复杂查询需用指令（详见下文）；
- **史莱姆区块查询**：显示当前坐标附近的史莱姆区块位置；
- **GSit动作**：触发坐下、躺下、爬行、旋转等趣味动作；
- **地图画生成**：上传图片，自动生成对应像素地图画；
- **排行榜**：查看死亡次数、在线时长、建筑规模等玩家排行数据（实时更新）；
- **爱心名单**：展示为服务器提供无偿捐赠的玩家名单（按捐赠时间排序）。

### `/kf`：区服切换

服务器支持多生存区服并行，输入 `/kf` 打开区服选择菜单，点击目标区服名称即可无缝切换（除创造服外，跨区服物品互通）。

| TPS状态 | 模拟距离 | 视野距离 |

| TPS≥19 | 7 | 10 |
| 18≤TPS＜19 | 5 | 8 |
| 16≤TPS＜18 | 4 | 7 |
| TPS＜16 | 3 | 6 |

## 💽 插件功能详解

所有插件功能均可通过 `/cd` 主菜单调用，也可直接输入指令使用，以下为关键插件的详细用法：

### CoreProtect（操作记录与回溯）

#### 功能定位

用于记录所有玩家的方块操作（放置、破坏、交互），支持回溯查询，帮助恢复被破坏建筑/物品，是服务器反破坏、纠纷处理的核心工具。

#### 核心指令

| 指令（简写） | 功能描述 | 示例（场景） |
|--------------|----------|--------------|
| `/co inspect`（`/co i`） | 启用/禁用检查模式<br>启用后右键方块可查看操作记录（玩家、时间、行为） | 发现基地箱子被拆 → 输入 `/co i` → 右键箱子位置 → 查看拆箱玩家信息 |
| `/co near`（`/co n`） | 查看当前位置附近10格内的方块操作记录 | 怀疑附近有恶意破坏 → 输入 `/co n` → 快速浏览近期操作 |
| `/co lookup`（`/co l`） | 精准过滤查询（按时间、玩家、行为） | 查找玩家“Steve”过去2小时在半径5格以内的挖掘记录 → `/co l t:2h p:Steve a:-block r:5` |

#### 常用过滤参数

- `t:时间`：如 `t:1h`（1小时）、`t:30m`（30分钟）、`t:2d`（2天）；
- `p:玩家ID`：指定查询某玩家的操作；
- `a:行为`：`a:+block`（放置）、`a:-block`（破坏）、`a:click`（交互，如打开箱子）；
- `r:半径`：如 `r:5`（查询5格内）。

### 地图画插件

#### 操作步骤

**请保证自己的物品栏拥有至少一个空地图！**

1. 输入 `/cd` → 选择“地图画”功能；
2. 根据提示，上传图片，输入相应指令，立即获得一个地图画。

#### 注意事项

- 按量生成地图画，避免占用过多服务器资源；
- 禁止生成违规内容图片，违者按公约处罚。

### 史莱姆区块查询插件

#### 用法
- 指令方式：输入 `/cd` → 选择“史莱姆区块查询”，或直接输入 `/scm`；
- 结果展示：箱子UI界面显示当前坐标附近的史莱姆区块；
- 适用场景：搭建史莱姆农场前定位刷怪区域，无需依赖种子查询工具。

### BreweryX（酿酒插件）

#### 功能简介

模拟真实酿酒流程（发酵→蒸馏→陈酿），可酿造啤酒、红酒、威士忌等饮品，特色酒品有独特效果（时间短，仅有娱乐作用）。

#### 完整酿酒流程

##### 第一步：发酵（基础酿造）

1. 准备工具：炼药锅（酿酒桶）、岩浆块/篝火（热源）、玻璃瓶；
2. 操作步骤：
   - 炼药锅装满水，下方放置热源方块（保持持续加热）；
   - 向炼药锅中投入配方原料（如小麦、甜浆果，详见配方表）；
   - 用钟右键炼药锅，查看剩余发酵时间；
   - 发酵完成后，用玻璃瓶右键炼药锅，获得发酵液。

##### 第二步：蒸馏（提升酒精度）

1. 准备工具：酿造台、萤石粉（催化剂，不消耗）、发酵液玻璃瓶；
2. 操作步骤：
   - 将发酵液玻璃瓶放入酿造台底层槽位；
   - 顶层槽位放入萤石粉，启动蒸馏；
   - 等待5分钟，蒸馏完成后获得蒸馏液（酒精度提升）。

##### 第三步：陈酿（赋予风味/效果）

1. 准备工具：木桶（原版/大型木制桶均可）、蒸馏液玻璃瓶；
2. 操作步骤：
   - 将蒸馏液玻璃瓶放入木桶中，关闭桶盖；
   - 陈酿时间按游戏内天数计算（1天=1年，详见配方表）；
   - 陈酿完成后，右键木桶取出成品酒（部分酒品会解锁特殊效果）。
