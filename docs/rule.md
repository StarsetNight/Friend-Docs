# 社区公约

## 作为玩家，你如何处理冲突（重要！）

虽然我们强调要以友谊为准则进行相处，但相处难免有小摩擦。别慌！按下面步骤解决，省时又省心。

首先，**不搞公开争吵**。不管啥矛盾，都别在大群、服务器聊天栏吵吵——公共区域是大家交流的地方，橙管看到会直接制止（比如禁言），越吵越难解决！

**能好好说话的话**，个人问题直接找对方私聊说清楚；社团之间有分歧，先找社长/负责人协商，九成问题这样就能化解。

最后，要是谈不拢，或者对方根本不配合，就加橙管群申请帮忙，但是**别在大群里直接喊管理**！

注意，橙管只看实际影响、公约规定和社区利益，**谁违规就处理谁**，哪怕是举报人自己也有问题，照样一视同仁！

## 游玩注意事项

**飞行、矿透、鞘翅加速（含 Tweakeroo、Lunar 等模组 / 客户端的加速功能）、种子逆向、自动挖矿、杀戮光环**这些作弊操作，坚决不能碰；但投影打印机、自动钓鱼、Disc Jockey（音符盒音乐）、灵魂出窍挖矿、矿物发光是允许用的，违规最高可永久封禁！

作为其他行为规范的前提，做事说话要平等友好，**别让多数玩家觉得反感或造成实质性破坏**，否则会根据情况处置。

**不得强制PVP**！特别注意的是，玩家**更不可以在玩家聚集的地区释放爆炸性、攻击性道具**（比如释放监守者），如有发现，严格处理，最高可永久封禁！

**禁止故意运作可使服务器产生严重卡顿的机器**，如有发现，严格处理，最高可永久封禁。

禁止制作或传播**违反中华人民共和国法律的、严重使人不适的、有恶意攻击性**的地图画、告示牌、成书（包括书与笔），一经发现，严肃处理，最高可永久封禁！

此外，即使是合法传播品，如涉及到个人、组织宣传信息，则**不得在非自愿的情况下暴力、强行传播给他人或其他基地**！

**禁止现实法定货币交易**服务器内游戏道具、游戏机器、游戏内职务、代肝等，如有发现，严格处理。

***“用人物，须明求；倘不问，即为偷。”***
不得未经同意使用、改动他人机器、建筑，拿取他人物品、非公共机器任意容器物品，一经发现即封。

被声明为原创建筑、机器的，就别盗用、抄袭，尊重他人的劳动付出。

老玩家不要打扰新人开荒体验，禁止主动向新人提供物资，**不得逼着新人加入某个社团或组织，更不能恶意争抢新人**。

## 群聊内、游戏内交流注意事项

进群后记得把**群昵称换成自己的游戏ID**，方便大家互相识别。

有疑问想找橙管，直接进橙管群说关键问题，**别在大群里反复喊，也别添油加醋说没用的**。

公共区域要守规矩，大群、服务器聊天栏是所有人的交流空间，各个年龄段的玩家都有，要注意！

- 不要刷屏（连续发 3 条及以上相同或没意义的消息），**不要浪费公共资源**；**不要发违反国家法律的文字、图片、视频**（比如暴力、恐怖、色情、不当政治内容）；
- 不要发让人**恶心不适、恶意攻击、地图炮歧视**的言论；
- **不误导**陌生人，尤其是新人（如装光影会封号），造成损失会被处罚；
- 另外，**不要在大群或服务器聊天栏吵架**，不管是出于什么原因，不得在公开环境吵架。

此外，请理解橙管工作。如果对橙管的处理有意见，私下聊或者去橙管群友好沟通，**不要公开指责橙管、故意曲解公约**——橙管也是人，他们会维护自己的声誉，反击损害声誉的行为。

## 社团事项

- 社团相处要和睦：每个社团都要管好自己人，社团之间要平等互助，**不要搞恶意争斗**，一定要遵守公约，一起维护公共环境；
- 社团信息要合规：社团的名字、宗旨、宣传内容，**都不能包含现实中的国家、政党、教派相关信息**。

## 附则

**本公约的最终解释权归Friend友谊服务器橙管组所有**；除服主修正决定外，橙管作出的处罚决定为服务器内最终处理结果，如处理事务不在本公约声明范围内，**请以处理结果为准**。
