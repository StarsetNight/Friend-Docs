# 附加严打条款

> [!WARNING] 本条款恢复执行
> 本条款自发布之日起正式生效，将持续执行至2026周目中期；若2026周目中期节点调整，执行截止时间同步顺延，具体以橙管公告为准。

## 制定目的与依据

为严厉整治Friend-友谊服务器内作弊行为泛滥的乱象，维护公平、健康的游戏环境，保障全体玩家的合法体验，根据橙管决议，特制定本临时附加条款，作为服务器现有规则的补充依据。

## 玩家连带责任处罚

1. 若玩家在游戏过程中观察到同行者（包括组队队友、社团成员等共同游戏对象）实施作弊行为，未及时进行劝阻（口头制止、向管理员举报等有效方式），经管理员核实后，对该玩家处以**警告处罚**；
2. 若查实玩家存在长期故意包庇作弊者的行为（包括隐瞒作弊事实、协助作弊者规避检测、伪造证据掩盖作弊行为等），视情节严重程度，对该玩家处以**6至12小时封禁处罚**；情节特别恶劣的，可延长封禁时长至24小时。

## 社团管理失职处罚

1. 若某社团内成员作弊行为频发且已形成猖獗态势，社团管理者（社长、副社长等核心管理人员）未采取任何有效制止措施（包括制定社团内部反作弊规则、移除作弊成员、向管理员报备作弊情况等），导致作弊行为长期存在并对服务器环境造成恶劣影响的，服主团队将对该社团处以**强制解散处罚**；
2. 被强制解散的社团，自解散之日起，需等待15天冷却期后方可重新提交建团申请，且重新申请需经服主团队严格审核。

## 术语定义

1. **作弊行为**：指使用第三方修改工具、游戏漏洞恶意获利、脚本自动化操作、透视/自瞄等违规插件、代打代练破坏公平性等行为，具体以服务器[《玩家公约》](./rule#第二章-违规处罚)为准；
2. **有效措施**：指能够**实际**遏制作弊行为、配合服务器管理的合理行动措施。

## 执行与申诉

1. 本条款由服务器管理员团队负责执行，处罚决定将通过服务器公告、玩家私信等方式通知当事人；
2. 当事人对处罚结果有异议的，可在收到处罚通知后24小时内，通过服务器申诉渠道（指定邮箱/申诉表单）提交申诉材料，服主团队将在48小时内复核并给出最终结论；
3. 申诉期间不停止处罚执行，复核确认处罚错误的，将立即解除处罚并消除相关记录。

## 附则

1. 本条款未明确规定的事项，参照服务器现有管理规则执行；
2. 本条款的最终解释权归Friend友谊服务器橙管组所有；
3. 若服务器后续发布新的反作弊相关规则，与本条款冲突的，以最新规则为准。
