# 举报与管理员功能

## ⛔ 举报功能

### 功能概括

针对服务器内违规玩家的投诉通道，提交举报后将触发对应处置流程，需配合提供有效证据。

### 命令用法

```Minecraft
/freport <玩家ID> <违规原因>
```
- 示例：`/freport Example_Player 轰炸友谊广场`
- 功能效果：提交举报后，被举报玩家将被强制下线；若被举报玩家发起申诉，橙管组将启动调查程序。

### 使用须知

1. 举报前需准备完整有效证据（如录屏、截图、聊天记录等），供申诉调查时核验；
2. 违规原因需如实填写，描述需清晰准确（例：“使用外挂作弊”“恶意破坏建筑”），不得虚构或夸大。

> [!WARNING] 禁止恶意举报
> 若以虚假信息、报复性目的等恶意举报他人，橙管组核实后有权给予举报人严重处罚（包括但不限于警告、短期封禁、永久封禁）。

## 💻 管理员操作功能

### 1. 解封玩家

- **功能概括**：解除被封禁玩家的限制状态
- **命令格式**：
```Minecraft
/funban <玩家ID>
```
- 说明：仅适用于已封禁且符合解封条件的玩家，操作后即时恢复账号正常使用权限。

### 2. 封禁玩家

- **功能概括**：对违规玩家实施限时或永久封禁处罚
- **命令格式**：
```Minecraft
/fban <玩家ID> <处罚时长> <违规原因>
```
- 说明：处罚时长需符合服务器规则，违规原因需详细记录，便于后续查询核实。

### 3. 更新封禁时间

- **功能概括**：调整已封禁玩家的处罚时长（延长或缩短）
- **命令格式**：
```Minecraft
/fup <玩家ID> <新处罚时长> <调整原因>
```
- 说明：调整需经橙管组内部确认，调整原因需明确（例：“补充违规证据，延长封禁”“主动认错且弥补损失，缩短封禁”）。

### 4. 监管模式

- **功能概括**：管理员隐身巡查、玩家信息查询及快速移动功能
- **命令格式**：
```Minecraft
/v
```
- 功能详情：
  1. 输入命令可切换监管模式的开启/关闭状态；
  2. 模式启用后：
     - 管理员自动进入隐身状态，右键点击玩家可查看其背包内容；
     - 双击`潜行键`（默认左Shift）可切换至旁观者模式，支持隐身飞行；
     - 在旁观者模式中，按下`中键`（鼠标滚轮按下）可激活传送模式，点击目标玩家即可传送至其身边。

### 通用规则注释

1. 处罚时长中，`-1` 代表永久封禁（例：`/fban Example_Player -1 严重作弊`）；
2. 限时封禁支持的时间格式：`1d`（1天）、`1h`（1小时）、`1m`（1分钟）、`1s`（1秒）；
3. 时间参数仅支持填写一种单位（不可同时混合填写“1d 2h”，需明确单一时长如“24h”）。

### 操作示例

```Minecraft
# 封禁玩家180天（原因：二次飞行作弊）
/fban Example_Player 180d 二次飞行作弊

# 更新玩家封禁时长为1小时（原因：暂未执行道歉）
/fup Example_Player 1h 暂未执行道歉

# 解封指定玩家
/funban Example_Player
```
