# 星级评定标准

## 星级评定

| 星级       | 周目在线时长 | 周目获得成就数 |
|------------|--------------|----------------|
| ⭐️ 一星     | 8小时       | 45（含解放末地）             |
| ⭐️⭐️ 二星    | 48小时      | 65             |
| ⭐️⭐️⭐️ 三星  | 120小时      | 85             |
| ⭐️⭐️⭐️⭐️ 四星 | 240小时      | 85             |
| ⭐️⭐️⭐️⭐️⭐️ 五星 | 600小时      | 85（全成就则无时长要求）            |

## 专属权益

| 星级       | 基础权限       | 附加专属权益     |
|------------|----------------|------------------------------------------------------------------------------|
| ⭐️ 一星     | 解除萌新限制、解锁地标（上限2）   | 敬请期待 |
| ⭐️⭐️ 二星    | 地标上限3              | 敬请期待 |
| ⭐️⭐️⭐️ 三星  | 地标上限4、广场权限   | 敬请期待 |
| ⭐️⭐️⭐️⭐️ 四星 | 地标上限5              | 敬请期待 |
| ⭐️⭐️⭐️⭐️⭐️ 五星 | 地标上限6              | 敬请期待 |

## 特别说明

`总在线时长累计超过1000小时，当前周目在线时长超过60小时，且当前周目获得的成就数超过60个`的玩家，可直接向服主提交权限申请，审核通过后即可解锁广场相关权限。
