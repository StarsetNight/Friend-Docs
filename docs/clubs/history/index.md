# 历史社团

## 概述

本选项卡收录服务器的**历史社团**，判定标准为：新周目开启后未完成重新登记流程的社团，均归入历史社团范畴。

历史社团包含两类情形：部分社团已完成正式解散流程；另一部分社团因失联（无法与管理者取得有效联系），且未完成新周目登记，自动划入本分类。

每个历史社团被归入的周目分类，代表其最后一次出现时所处的服务器周目。

## 为什么我的社团出现在本选项卡？

若您为社团管理者，无需过度担忧，可根据当前周目阶段对应处理：
- 若处于**周目末期**：系管理员（橙管）为新周目筹备提前调整公告分类，所有社团默认归类为历史社团，新周目开启后完成登记即可恢复正常分类；
- 若处于**周目初期/中期**：请立即通过橙管群联系管理员反馈，核实社团登记状态并完成补登记。
