# 社团

欢迎来到Friend友谊服务器社团列表！

## 关于社团

社团是本服务器玩家集体活动的基本单位，也是玩家多方面发展的特色所在。通过社团，玩家可以：

- 参与集体活动和项目
- 发展多元化兴趣和技能
- 建立友谊和交流

我们致力于发展友谊，让每位玩家在社团中找到属于自己的归属感。

每一个社团，都是一条重要的音轨，它们共同奏成了本服务器的交响曲。
