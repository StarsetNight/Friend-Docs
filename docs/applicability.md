# 谁适合加入，而谁又不适合加入
Friend-友谊服务器是由管理团队与核心玩家共同搭建的优质游戏社区，始终以“友好协作、沉浸式体验”为核心价值观，致力于为匹配社区定位的玩家提供稳定、和谐的游戏环境。以下明确适合与不适合加入本服务器的玩家类型，供各位参考：

## 适合加入的玩家

1. **社交型玩家**：乐于在游戏中拓展社交圈，主动与他人互动、交流，渴望结识志同道合的游戏伙伴；
2. **合作型玩家**：偏好组队游玩但缺乏联机经验，愿意学习协作技巧，期待在团队中共同完成目标；
3. **建筑爱好者**：痴迷于游戏内的建筑创作，无论是精致小屋还是宏大建筑群，愿意分享作品、交流设计思路；
4. **休闲生存玩家**：热爱慢节奏的生存体验，享受探索、采集、建造的过程，不追求功利化的游戏进度。

## 不适合加入的玩家

1. **缺乏尊重意识的玩家**：素质低下，频繁使用不文明用语，轻视或攻击其他玩家，破坏社区和谐氛围；
2. **低龄玩家（12 岁及以下）**：服务器核心玩家群体年龄集中在 13-25 岁区间，12 岁及以下玩家不仅大概率难以找到同龄玩伴，更可能因交流语境、话题偏好与核心群体存在差异，无法适应社区的互动节奏与沟通模式；且低龄玩家的游戏行为习惯、规则理解能力相对薄弱，易与其他玩家产生体验冲突。
3. **过度依赖即时反馈的玩家**：对社群互动、组队响应有过高期待，无法接受非即时性的沟通模式，易因等待产生负面情绪；
4. **沉默寡言的独行玩家**：全程拒绝交流，习惯独自游玩且不愿参与集体活动，难以融入服务器的协作氛围；
5. **极致生电需求的玩家**：服务器侧重生存与社交体验，未针对高效生产、红石专精做特殊优化，无法满足硬核生电玩法需求；
6. **作弊违规玩家**：习惯使用外挂、利用漏洞获利，或通过违规手段破坏游戏公平性；
7. **恶意破坏玩家**：以摧毁他人建筑、干扰他人游戏进程为乐，无视社区规则与他人体验；
8. **功利主义的急躁玩家**：过度追求游戏“结果”（如快速通关、资源囤积），排斥慢节奏体验，无法享受探索与创造的过程。
